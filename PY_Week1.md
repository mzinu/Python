## 파이썬 기초  

2.3.1 시맨틱
    - 콜론(:) : 코드 블록의 시작
    - 세미콜론(;) 끝낼 필요 없음. 한 줄에서 여러 문장 구분할 때는 사용 -> 지양
    - 주석 : # 뒤
    - 모든 객체는 특정한 자료형(or 클래스)을 가짐
    - isintance(a, int) : a가 int 자료형인지 검사하는 함수
    - 모듈 : .py 파일
    - 동일 객체 참조 검사 : is/is not

2.3.2 스칼라 자료형
    - 문자형은 ' or " 로 둘러싸서 표현가능
    - 여러줄 문자형은 세개의 ' or "
    - 리스트로 문자열 변경 불가능 -> .replace() 사용
    - format 메서드
        - {0:.2f} : 첫번째 인수를 소수점 아래 두자리까지만 float 출력
        - {1:s} : 두번째 인수를 문자열로 포맷
        - {2:d} : 세번째 인수를 정수로 포맷
    - 형변환함수 : str, bool, int, float
    - None == null

2.3.3 제어 흐름
    - range() : 균일한 간격의 연속된 정수 반환, 간격 음수 가능
